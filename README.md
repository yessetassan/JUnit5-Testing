                                          F.I.R.S.T Principle

1. Fast - Unit Tests run fast

2. Independent - Unit Tests are independent

3. Repeatable - Unit tests are repeatable

4. Self-validating - Unit Tests validates itself

5. Thorough & Timely - Cover Edge Cases
